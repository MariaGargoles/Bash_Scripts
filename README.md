# Scripts de Ciberseguridad y Gestión de Usuarios para Linux

## Descripción

Este repositorio contiene una colección de scripts en Shell para mejorar las medidas de ciberseguridad en sistemas Linux, junto con scripts para la gestión de usuarios. Estos scripts tienen como objetivo automatizar varias tareas de seguridad y simplificar el proceso de creación y gestión de cuentas de usuario.
